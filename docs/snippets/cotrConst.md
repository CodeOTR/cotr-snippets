title: cotrConst

Creates a constant variable

**Aliases**:
- cotrConst
- constant
- cotrVal

## cpp
```cpp
const type myConst = value;
```

## csharp
```csharp
const Type myConst = value;
```

## dart
```dart
const myConst =2;
```

## go
```go
const variableName variableType = variableValue
```

## haskell
```haskell
-- Haskell does not have a 'const' keyword. Use 'let' for immutable values.
```

## java
```java
final Type myConst = value;
```

