title: cotrEntry
description: The cotrEntry snippet.

Describes the entry point of a TypeScript application.

**Aliases**:
- cotrEntry
- entry
- start
- startingPoint

## cpp
```cpp
// C++ Entry Point
// To compile and run this program, use: `g++ filename.cpp -o output && ./output`

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```
