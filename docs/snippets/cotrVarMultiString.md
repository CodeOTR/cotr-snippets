Creates a multi-line string variable

**Aliases**:
- cotrVarMultiString
- multiString
- variableMultiString
- multiStringVariable
- multilineText

## go
```go
var variableName = `
Line 1
Line 2
Line 3
`
```

## java
```java
String myString =
    "Line 1" +
    "\nLine 2" +
    "\nLine 3";
```

```javascript
`;
```
